Utilizaremos uma aplicação chamada vercel.com (https://vercel.com/ - https://vercel.com/login). Ela aponta para um dos nossos repositórios do GitHub e

link https://jogo-secreto-amber.vercel.app/

Vercel:
A Vercel, por sua vez, é uma plataforma de hospedagem e implantação de aplicativos da web moderna, especializada em
oferecer experiências de carregamento rápido e desempenho otimizado. Ela é frequentemente utilizada para hospedar
sites estáticos e aplicações de página única (ou SPAs), aproveitando técnicas avançadas e eficiente aos usuários
finais.
